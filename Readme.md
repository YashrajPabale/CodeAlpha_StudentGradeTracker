# 🎓 Student Grade Tracker

## 📌 Overview

The Student Grade Tracker is a Java-based console application designed to manage student records and analyze their academic performance.

---

## 🚀 Features

* Add student name and marks
* Store data using ArrayList
* Calculate:

  * Average marks
  * Highest marks
  * Lowest marks
* Display student report

---

## 🛠️ Technologies Used

* Java
* ArrayList
* OOP Concepts
* Console-based interface

---

## ▶️ How to Run

### Compile:

```
javac StudentGradeTracker.java
```

### Run:

```
java StudentGradeTracker
```

---

## 📊 Sample Output

* Student list with marks
* Average score
* Highest & lowest marks
